# Рішення проблеми зі зв'язком Telegram

## Проблема
Коли ви виходите з дому, Telegram бот втрачає зв'язок, бо працює на вашому домашньому комп'ютері.

## Рішення 1: SSH тунель (рекомендовано)

### Крок 1: Встановіть Termius на телефон
- Завантажте Termius з App Store (iOS) або Google Play (Android)
- Створіть обліковий запис (безкоштовно)

### Крок 2: Налаштуйте SSH підключення
1. Відкрийте Termius
2. Додайте новий хост:
   - **Hostname:** `localhost` (або IP вашого домашнього ПК, якщо знаєте)
   - **Username:** `oleg` (ваше ім'я користувача)
   - **Password:** (пароль вашого ПК)
   - **Port:** `22`

### Крок 3: Налаштуйте порт-форвардинг
У Termius:
1. Відкрийте налаштування хоста
2. Перейдіть до "Port Forwarding"
3. Додайте нове правило:
   - **Type:** Local
   - **From Port:** `3001` (або будь-який вільний порт)
   - **To Host:** `localhost`
   - **To Port:** `3001`

### Крок 4: Підключіться та запустіть тунель
1. Підключіться до хоста
2. Увімкніть порт-форвардинг
3. Тепер ваш телефон може звертатися до `http://localhost:3001` як до локального сервера

### Крок 5: Налаштуйте Telegram бота
1. Створіть бота через @BotFather
2. Отримайте токен
3. Вставте токен у файл `wolf-eggs-game/bot/.env`
4. Запустіть бота: `cd wolf-eggs-game/bot && npm start`

## Рішення 2: VPS хостинг ($5-10/місяць)

### Переваги:
- 24/7 доступність
- Публічний IP
- Не залежить від домашнього інтернету

### Кроки:
1. **Обрати провайдера:**
   - DigitalOcean ($5/місяць)
   - Hetzner ($4.51/місяць)
   - AWS Lightsail ($3.5/місяць)

2. **Створити сервер:**
   - OS: Ubuntu 22.04
   - RAM: 1GB
   - SSD: 25GB

3. **Розгорнути гру:**
   ```bash
   # Клонувати проект
   git clone <ваш репозиторій>
   
   # Встановити Docker
   sudo apt install docker.io
   
   # Запустити PostgreSQL
   docker run --name postgres -e POSTGRES_PASSWORD=password -d -p 5432:5432 postgres
   
   # Запустити бекенд
   cd wolf-eggs-game/backend
   npm install
   npm start
   
   # Запустити бота
   cd ../bot
   npm install
   npm start
   ```

4. **Налаштувати домен та SSL:**
   - Купити домен ($10/рік)
   - Налаштувати Cloudflare (безкоштовно)
   - Видати SSL сертифікат

## Рішення 3: Веб-чат OpenClaw (вже працює)

### Поточний статус:
- OpenClaw gateway доступний за адресою: `http://127.0.0.1:18789/`
- Токен: `59805d150d77e3608372b32325e4444617c73a993b6c5594`
- Контрольна панель: `http://127.0.0.1:18789/#token=59805d150d77e3608372b32325e4444617c73a993b6c5594`

### Як отримати доступ ззовні:
1. **Налаштуйте SSH тунель для порта 18789:**
   ```bash
   ssh -R 18789:localhost:18789 ваш_сервер
   ```

2. **Або використовуйте ngrok (безкоштовно):**
   ```bash
   ngrok http 18789
   ```

3. **Після цього відкрийте веб-чат за публічною URL**

## Термінове рішення (поки)

### Використовуйте веб-чат OpenClaw через SSH тунель:
1. Встановіть Termius
2. Налаштуйте порт-форвардинг для порта 18789
3. Відкрийте у браузері телефону: `http://localhost:18789`
4. Введіть токен з MEMORY.md

### Команда для SSH тунелю на ПК:
```bash
ssh -N -R 18789:localhost:18789 user@ваш_сервер
```
